## Repository Purpose

This repository demonstrates production usage patterns for stellar-react-native-wallet-kit.

It should remain simple, educational, and easy to understand.

## Contributor Guidelines

When adding examples:

* Prefer clarity over abstraction
* Minimize complexity
* Explain important decisions
* Include comments when needed

## Quality Standards

Every example should:

* Compile successfully
* Follow SDK best practices
* Demonstrate real-world usage
* Be easy for beginners to follow
