# stellar-rn-wallet-kit-example

Reference application demonstrating real-world usage of stellar-react-native-wallet-kit.

## Purpose

This repository acts as:

* Integration test environment
* Learning resource
* Contributor sandbox
* SDK showcase

Developers can use this application to understand best practices for Stellar mobile development.

---

## Demonstrated Features

### Wallets

* Connect Freighter Mobile
* Disconnect wallets
* Session persistence

### Accounts

* Load balances
* Display assets
* Refresh state

### Transactions

* Sign transactions
* Submit transactions
* Error handling

### Soroban

* Contract invocation
* Result decoding
* Simulation workflows

---

## Running Locally

```bash
yarn install
yarn start
```
